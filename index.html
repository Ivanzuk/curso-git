Curso Git - Alura

Trabalhando em Equipe com Controle e Segurança

Um curso que explora os benefícios e praticidades de utilizar o Git como ferramenta de controle de versão para projetos em qualquer linguagem, em qualquer plataforma.

Principais beneficios:

  • Funciona de maneira distribuída
  • Permite a edição concorrente de arquivos do projeto
